Termos da HTML

Dica: Pressione Ctrl + F para localiar a tag ou palavra que deseja encontar!

HTML
Termo - ing.
Hypertext Markup Language. Traduzido para o Português, Linguagem Marcação Hipertexto. É uma linguagem usada na criação de sites. Sua extensão pode ser .htm ou .html
<h1>
Tag
<h1> é uma tag em HTML que cria um título de primeiro nível.
Exemplo: 
<h1>
